# NeedThingsDone

Organized project structure.

## Run locally
Open `index.html` with Live Server.

## Structure
- `index.html` — home page
- `pages/` — public site pages
- `onboarding/` — signup/account/profile setup flow
- `assets/images/` — images such as Otto
- `css/style.css` — imports the CSS files
- `css/legacy.css` — current site styles preserved safely
- `js/` — site scripts
- `backend/` — Supabase setup

## Notes
The site paths were updated for the new folders. Future CSS should be added to the smaller files instead of growing `legacy.css`.
